# E-N-C 캠퍼스 안전 시스템: 웹페이지 기획안

이 문서는 'E-N-C 데이터 기반 캠퍼스 안전 시스템' 프로젝트의 웹페이지 기능 및 메뉴 구성을 정의하고, 개발 진행 상황을 추적하기 위해 작성되었습니다.

## 1. 웹페이지 컨셉
- [ ] **데이터 중심의 실시간 관제 대시보드**

## 2. 주요 메뉴 구성
- [ ] **대시보드 (Dashboard):** 전체 상황 요약
- [ ] **상세 분석 (Analysis):** 특정 카메라 심층 분석
- [ ] **지도 (Map):** 공간 기반 위험도 파악

## 3. 메뉴별 세부 기능 계획

### 3.1. 대시보드 (Dashboard)
- [x] **화면 구성:** 등록된 모든 CCTV를 '카메라 카드' 형태의 그리드로 표시
- [x] **카드 기능: 실시간 영상/스냅샷 표시**
    - [x] AI가 탐지한 객체에 바운딩 박스 오버레이
- [x] **카드 기능: 현재 밀집도 표시**
    - [x] 밀집도 수치 (예: 0.65)
    - [x] 위험 상태 (여유/주의/위험)를 색상으로 표시
- [x] **카드 기능: 예측 정보 표시 (핵심 기능)**
    - [x] 밀집도 변화율(속도) 표시 (예: +5%/min ▲)
    - [x] 밀집도 가속도 표시
- [x] **카드 기능: 위험 지속 시간 타이머 표시**
- [x] **전체 알림: '최신 알림' 목록 표시**
    - [x] 예측 기반 경보 메시지를 시간순으로 표시

### 3.2. 상세 분석 (Analysis)
- [ ] **화면 구성:** 상단 (선택된 카메라의 대형 영상), 하단 (분석 차트 및 데이터)
- [ ] **기능: 분석 레이어 선택 기능 (핵심 기능)**
    - [ ] `[ ] 객체 탐지` 레이어 On/Off
    - [ ] `[ ] 동적 ROI` 레이어 On/Off
    - [ ] `[ ] 실시간 히트맵` 레이어 On/Off
- [ ] **기능: 시계열 패턴 분석 차트**
    - [ ] 시간 범위 선택 기능 (지난 1시간, 오늘, 이번 주 등)
    - [ ] **패턴 오버레이 기능:** '오늘' 데이터 위에 '어제', '주중 평균' 등 데이터 겹쳐보기
    - [ ] **이상 패턴 탐지 기능:** 통계적 '정상 범위'를 밴드 형태로 표시하여 이상 현상 강조
- [ ] **기능: 히트맵 캘린더**
    - [ ] 달력 위에 시간대별 평균 밀집도를 색상으로 표시하여 주기적 패턴 분석

### 3.3. 지도 (Map)
- [ ] **화면 구성:** 캠퍼스 지도 위에 CCTV 위치를 마커로 표시
- [ ] **기능: 실시간 위험도 연동**
    - [ ] 마커 색상을 실시간 밀집도 수준(여유/주의/위험)에 따라 동적으로 변경
- [ ] **기능: 요약 정보 팝업**
    - [ ] 마커 클릭 시, 해당 카메라의 요약 정보(이름, 썸네일, 밀집도 값) 팝업 표시
    - [ ] 팝업 내 '상세 분석' 페이지로 이동하는 링크 제공
